# 📚 MAGSHOP ERP - ΕΚΠΑΙΔΕΥΤΙΚΟ ΠΛΑΝΟ

## 🎯 Στόχος

Να φτιάξουμε το MagShop ERP **από το μηδέν**, με εκπαιδευτικό τρόπο, γραμμή-γραμμή, εξηγώντας κάθε κομμάτι κώδικα.

---

## 📦 Δομή Αρχείων (Νέα)

```
magshop-erp/
├── index.html          (Κύρια σελίδα)
├── css/
│   └── style.css       (Όλα τα styles)
├── js/
│   └── app.js          (Όλη η λογική - θα χωρίσουμε αργότερα)
└── assets/
    └── logo/
```

> **Σημείωση:** Ξεκινάμε με απλή δομή. Καθώς μαθαίνεις, θα χωρίζουμε σε modules!

## 🎓 Γιατί είναι Σωστό αυτό το Πλανό;

### Ως Study Case:

| Οφέλη              | Εξήγηση                           |
| ------------------ | --------------------------------- |
| ✅ Version Control | Μαθαίνεις Git βήμα-βήμα           |
| ✅ Portfolio       | Έχεις 10+ commits στο GitHub      |
| ✅ Real Project    | Δουλεύει σε πραγματικό κατάστημα! |
| ✅ Debugging       | Μαθαίνεις να βρίσκεις λάθη        |
| ✅ Confidence      | Κάθε μέρα ολοκληρώνεις κάτι       |

### Ως Εκπαίδευση:

- **Κάθε version** = Συγκεκριμένη δεξιότητα
- **Δυσκολία** = Προοδευτική αύξηση
- **Εξήγηση** = Κάθε γραμμή εξηγείται
- **Επανάληψη** = Κάνουμε repeat με νέες λειτουργίες
